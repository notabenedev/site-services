# Site reviews

Интерфейс для создания Услуг с галлереей. После создания у Услуги есть вкладки "Просмотр", "Галлерея" и "Метатеги". 

## Установка

    php artisan migrate

    php artisan make:services {--all : Run all}
                          {--menu : Config menu}
                          {--models : Export models}
                          {--controllers : Export controllers}
                          {--config : Make config}
                          {--only-default : Create default rules}
                          {--policies : Export and create rules}
